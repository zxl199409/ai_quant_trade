## 🔧 使用真实数据

目前使用的是**模拟数据**。要使用真实A股数据，有3种方式：

### 方式1：使用 AKShare（免费，推荐）

```bash
# 1. 安装 AKShare
pip install akshare

# 2. 下载数据（Python代码）
import akshare as ak
import pandas as pd

# 下载平安银行
df = ak.stock_zh_a_hist(symbol="000001",
                        start_date="20240101",
                        end_date="20251027")

# 重命名列以匹配程序格式
df = df.rename(columns={
    '日期': 'trade_date',
    '开盘': 'open',
    '最高': 'high',
    '最低': 'low',
    '收盘': 'close',
    '成交量': 'vol'
})

# 保存
df.to_csv('./data/000001.SZ.csv', index=False)
```

### 方式2：使用通达信导出

1. 在通达信中打开股票K线图
2. 右键 → 数据导出 → 导出所有数据
3. 保存为CSV格式
4. 将文件重命名为 `000001.SZ.csv` 并放到 `data/` 目录

### 方式3：使用 Tushare（需要积分）

需要注册并获取token：https://tushare.pro

## 📈 理解回测结果

运行后会显示：

### 关键指标
- **收益率**: 策略的总体表现
- **胜率**: 盈利交易占比
- **盈亏比**: 平均盈利/平均亏损
- **交易次数**: 信号触发频率

### 交易明细
- 每笔买入卖出的价格、数量
- 盈亏金额和百分比
- 卖出原因（止盈/止损）

## 💡 优化建议

如果回测效果不理想：

1. **调整参数**
   - 尝试不同的止盈止损比例
   - 修改布林带周期（10/20/30）

2. **测试更多股票**
   - 策略可能在不同股票上表现差异大
   - 建议测试至少10只股票

3. **更换时间段**
   - 牛市和熊市表现会有差异
   - 可以分段测试

## ❓ 常见问题

### Q: 为什么没有交易信号？

A: 双信号条件比较严格，可能出现以下情况：
- 时间范围太短
- 策略条件过于严格
- 数据质量问题

**解决方法**:
1. 延长回测时间（如测试2年数据）
2. 测试更多股票
3. 适当放宽条件（如紫色信号改为2%）

### Q: 显示 "未找到quant_brain模块"？

A: 这是正常的！程序已经做了兼容处理，会使用简化版回测，不影响使用。

### Q: 收益率为负？

A: 这说明策略在该时段表现不佳，这是正常的。可以：
- 优化参数
- 测试其他时间段
- 分析失败原因改进策略
