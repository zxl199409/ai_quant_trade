# 通达信做T指标系统 - 完整使用说明

## 📦 指标清单

本套做T指标系统包含3个核心指标：

1. **分时做T主图** - 分时图叠加指标，显示高低点、买卖区间
2. **资金流向监控** - 副图指标，监控主力资金进出
3. **做T买卖点提示** - 副图指标，综合评分给出精准买卖点

---

## 🔧 安装步骤

### 方法一：公式管理器导入（推荐）

1. **打开通达信软件**

2. **进入公式管理器**
   - 快捷键：`Ctrl + F`
   - 或菜单：功能 → 公式系统 → 公式管理器

3. **导入分时做T主图**
   - 选择"技术指标公式" → "分时指标"
   - 点击"新建"
   - 公式名称：`分时做T主图`
   - 公式描述：`分时做T辅助指标，显示高低点和买卖区间`
   - 复制 `分时做T主图.txt` 的全部内容到公式编辑器
   - 点击"确定"保存

4. **导入资金流向监控**
   - 选择"技术指标公式" → "其他类型"
   - 点击"新建"
   - 公式名称：`资金流向监控`
   - 公式描述：`监控主力资金和散户资金流入流出`
   - 复制 `资金流向监控.txt` 的全部内容到公式编辑器
   - 点击"确定"保存

5. **导入做T买卖点提示**
   - 选择"技术指标公式" → "其他类型"
   - 点击"新建"
   - 公式名称：`做T买卖点提示`
   - 公式描述：`综合评分系统，精准提示买卖点`
   - 复制 `做T买卖点提示.txt` 的全部内容到公式编辑器
   - 点击"确定"保存

### 方法二：直接导入文件

1. 将三个 `.txt` 文件复制到通达信安装目录下的 `T0002\blocknew` 文件夹
2. 重启通达信软件
3. 在公式管理器中可以看到新导入的指标

## 📊 使用方法

### 一、分时做T主图的使用

#### 1. 加载指标

- 打开股票分时图（快捷键：`05` + `Enter`）
- 在分时图上右键 → 选择"叠加指标"
- 找到"分时做T主图"并选择
- 指标会叠加在分时图上

#### 2. 指标说明

**核心元素：**

| 元素 | 颜色 | 说明 |
|------|------|------|
| **红色区域** | 红色填充 | 卖出区：价格进入此区域考虑卖出 |
| **绿色区域** | 绿色填充 | 买入区：价格进入此区域考虑买入 |
| **向上箭头** | 红色 | 买入信号：出现此信号可以买入 |
| **向下箭头** | 绿色 | 卖出信号：出现此信号可以卖出 |
| **黄色线** | 黄色 | 均价线：重要参考线 |
| **白色虚线** | 白色 | 昨收价、压力位、支撑位 |

**操作要点：**

1. **买入时机**
   - 价格回落到绿色买入区
   - 出现向上箭头买入信号
   - 价格回踩均价线并企稳
   - 显示"低点"文字标注

2. **卖出时机**
   - 价格上涨到红色卖出区
   - 出现向下箭头卖出信号
   - 价格远离均价线1%以上
   - 显示"高点"文字标注

3. **观望时机**
   - 价格在买卖区间之间
   - 没有明确的箭头信号
   - 价格紧贴均价线震荡

#### 3. 实战案例

**案例1：标准倒T操作**
```
09:45  价格154.5元进入绿色区域，出现买入信号 → 买入2手
10:30  价格158.8元进入红色区域，出现卖出信号 → 卖出2手
盈利：(158.8 - 154.5) × 200股 = 860元
```

**案例2：回踩均价线**
```
10:15  价格从上方回踩均价线，出现买入信号 → 买入2手
11:00  价格反弹至卖出区，出现卖出信号 → 卖出2手
```

### 二、资金流向监控的使用

#### 1. 加载指标

- 在分时图下方的副图区域
- 右键 → 选择指标 → 找到"资金流向监控"
- 或直接输入指标代码快速加载

#### 2. 指标说明

**核心指标线：**

| 指标线 | 颜色 | 含义 |
|--------|------|------|
| **红色柱** | 红色 | 主力资金流入 |
| **绿色柱** | 绿色 | 主力资金流出 |
| **橙色线** | 橙色 | 主力累计净流入（最重要）|
| **紫色线** | 紫色 | 主力强度（15分钟）|
| **青色线** | 青色 | 买卖力道 |
| **白色线** | 白色 | 零轴（正负分界线）|

**特殊信号：**

- **★大单异动★** - 超大单突然涌入，高度关注
- **▲资金加速流入** - 主力资金加速买入，可以跟进
- **▼资金加速流出** - 主力资金加速卖出，注意风险
- **顶背离** - 价格新高但资金流出，准备卖出
- **底背离** - 价格新低但资金流入，准备买入

#### 3. 操作策略

**买入信号组合：**
1. 主力强度由负转正（柱子变红）
2. 主力累计净流入上升
3. 出现"买"字标注
4. 配合分时做T主图的买入信号

**卖出信号组合：**
1. 主力强度由正转负（柱子变绿）
2. 主力累计净流入下降
3. 出现"卖"字标注
4. 配合分时做T主图的卖出信号

**高级技巧：**

1. **背离交易**
   - 底背离出现：价格创新低但资金流入 → 大胆买入
   - 顶背离出现：价格创新高但资金流出 → 果断卖出

2. **资金异动跟随**
   - 看到"大单异动"提示 → 立即关注
   - 资金加速流入 → 可以追高买入
   - 资金加速流出 → 及时止盈

3. **趋势确认**
   - 主力累计净流入持续上升 → 可以持股
   - 主力累计净流入持续下降 → 不宜做多

### 三、做T买卖点提示的使用

#### 1. 加载指标

- 在分时图下方再添加一个副图区域
- 右键 → 选择指标 → 找到"做T买卖点提示"
- 建议将此指标放在资金流向监控下方

#### 2. 指标说明

**评分系统：**

这是一个0-100分的智能评分系统，综合考虑：
- 价格位置（30分）
- 与均价线关系（25分）
- 量能配合（20分）
- 连续涨跌（15分）
- 技术形态/盈利空间（10分）

**信号等级：**

| 评分 | 信号等级 | 显示 | 操作建议 |
|------|---------|------|---------|
| 80-100分 | ★强烈买入★ | 红色大柱 + 星号 | 立即买入，机会难得 |
| 60-79分 | 买入信号 | 粉色柱 + 箭头 | 可以买入，胜率较高 |
| 40-59分 | 关注买入 | 黄色柱 | 关注，等待更好时机 |
| 0-39分 | 观望 | - | 不宜操作 |

**负分同理：**
- -80到-100分：★强烈卖出★
- -60到-79分：卖出信号
- -40到-59分：关注卖出

**可视化显示：**

- **红色柱子** = 买入力量（越高越强）
- **绿色柱子** = 卖出力量（越低越强）
- **白色曲线** = 做T综合指标
- **黄色虚线** = 做T指标均线（趋势）

#### 3. 操作策略

**标准操作流程：**

1. **观察评分**
   - 看左上角显示的实时评分
   - 买入评分 > 60分 → 准备买入
   - 卖出评分 > 60分 → 准备卖出

2. **等待信号**
   - 买入评分达到80分 → 立即买入
   - 买入评分60-79分 → 可以买入
   - 结合其他两个指标确认

3. **执行交易**
   - 看到"★强烈买入★" → 不要犹豫，立即买入
   - 看到"★强烈卖出★" → 不要贪心，立即卖出
   - 看到"可以买入/卖出" → 根据自己风格操作

**高级技巧：**

1. **趋势判断**
   - 看右上角"短期趋势：偏多/偏空"
   - 偏多时：优先考虑买入信号，谨慎卖出
   - 偏空时：优先考虑卖出信号，谨慎买入

2. **交叉信号**
   - 白色曲线从下向上穿越零轴 → 买入信号
   - 白色曲线从上向下穿越零轴 → 卖出信号

3. **统计数据参考**
   - 看底部"今日买入信号X次"
   - 信号次数多 → 震荡行情，适合做T
   - 信号次数少 → 单边行情，谨慎做T

## ⚠️ 注意事项

### 1. 风险控制

- ✅ **严格止损**：买入后跌2%立即止损
- ✅ **仓位管理**：T仓不超过底仓的75%
- ✅ **及时止盈**：有2-3元利润就卖，不贪心
- ❌ **不追涨杀跌**：没有信号不操作
- ❌ **不逆势操作**：趋势向下不做多

### 2. 信号冲突处理

**如果三个指标信号不一致：**

优先级排序：
1. **资金流向监控**（最重要）- 主力资金决定方向
2. **做T买卖点提示** - 综合评分系统
3. **分时做T主图** - 价格位置参考

**处理原则：**
- 至少2个指标同向才操作
- 资金流向与其他指标冲突时，以资金流向为准
- 信号不明确时选择观望

### 3. 特殊情况应对

**大盘暴跌**
- 停止做T，保护底仓
- 如果跌破20日线，考虑止损

**突发利好/利空**
- 暂停按指标操作
- 观察消息面影响
- 等待走势稳定后再操作

**连续做T失败**
- 失败2次以上，停止做T
- 反思操作失误
- 第二天重新开始

### 4. 指标参数优化

如果发现信号不够准确，可以调整参数：

**分时做T主图：**
- `N:=30;` → 调整为20-40，控制高低点敏感度

**资金流向监控：**
- `N1:=15;` → 调整为10-20，控制资金强度周期

**做T买卖点提示：**
- 调整各项评分权重，适应不同风格

   - 9:40-10:30：最佳买入时段（早盘回调）

   - 缩量下跌 + 放量反弹 = 最佳买点
   - 放量上涨 + 缩量滞涨 = 最佳卖点

   - 恐慌性下跌到买入区 = 大胆买入
   - 兴奋性上涨到卖出区 = 果断卖出

### 常见误区

❌ **错误1：频繁操作**
- 一天做T 5次以上，手续费吃掉利润
- 正确：每天1-2次高质量操作

❌ **错误2：不设止损**
- 买入后继续下跌不止损，扩大亏损
- 正确：跌破-2%立即止损

❌ **错误3：死守T仓**
- T仓没卖出，隔夜持有增加风险
- 正确：尽量日内清仓，不隔夜

❌ **错误4：只看一个指标**
- 只看价格位置就操作，忽略资金面
- 正确：三个指标综合判断

❌ **错误5：贪心不止盈**
- 有3元利润不卖，等到5元，结果回吐
- 正确：2-3元利润就卖，积少成多

## 📞 常见问题解答（FAQ）

**Q1：三个指标必须全部使用吗？**
A：建议全部使用，相互验证提高成功率。如果只用一个，推荐"做T买卖点提示"。

**Q2：指标信号滞后怎么办？**
A：分时指标有一定滞后性，建议结合盘口实时数据。强烈信号出现时要快速反应。

**Q3：开盘就涨停或跌停怎么办？**
A：极端行情不适合做T，停止操作观望。

**Q4：成交量很小的股票能用吗？**
A：不建议。日成交额最好在10亿以上，流动性好才适合做T。

**Q5：指标在通达信手机版能用吗？**
A：可以，但显示效果不如电脑版。建议用电脑操作。

**Q6：可以用于其他股票吗？**
A：可以，但要调整参数。活跃股票效果更好。

**Q7：指标公式能否自动交易？**
A：这些指标仅提供信号，不能自动下单。需要手动操作。

**Q8：信号太多如何筛选？**
A：只操作"强烈买入/卖出"信号（评分80分以上），提高胜率。

## 💾 技术支持

### 指标优化建议

如果你有以下需求，可以考虑自行调整：

1. **提高信号灵敏度**
   - 降低买卖评分阈值（80分→70分）
   - 缩短周期参数（30→20）

2. **降低虚假信号**
   - 提高买卖评分阈值（80分→90分）
   - 延长周期参数（30→40）

3. **适应不同股票**
   - 活跃股票：缩短周期，降低阈值
   - 稳健股票：延长周期，提高阈值

### 数据要求

- 通达信需要订阅Level-2数据才能显示准确的资金流向
- 如果没有Level-2，资金流向指标会基于估算，准确度会降低
- 建议使用实时行情，延时行情效果会打折扣
